# 📈 Professional Stock Analysis Platform v2.0

## 🎯 系統概述

這是一個機構級的 AI 股市分析系統，集合了高等技術分析、量化評分、風險管理與時間序列預測於一身。適合專業投資者、基金經理與量化分析師使用。

### 🌟 核心特性

#### 1. **技術分析模塊**
- **布林通道** (Bollinger Bands): 識別超買超賣區域
- **MACD** (Moving Average Convergence Divergence): 動能與趨勢變化檢測
- **RSI** (Relative Strength Index): 相對強度指數分析
- **ATR** (Average True Range): 波幅衡量
- **成交量分析**: 價量背離信號

#### 2. **量化評分系統**
```
綜合技術評分 = 
  趨勢評分(35%) + 動能評分(30%) + RSI評分(20%) + 成交量評分(15%)
```

評分範圍：0-100
- 70+ : 強勢買進信號
- 60-70 : 適度買進
- 40-60 : 持股觀望
- 30-40 : 考慮減碼
- <30 : 強勢賣出

#### 3. **AI 預測引擎**
- **Prophet 模型**: Facebook 開發的時間序列預測
  - 自動檢測趨勢與季節性
  - 95% 置信區間
  - 支援 30-365 天超期預測

#### 4. **風險管理系統**
| 指標 | 說明 |
|-----|------|
| **年化波動率** | 衡量股價波動程度 |
| **最大回撤** | 歷史最大虧損幅度 |
| **夏普比率** | 風險調整後的報酬率 |
| **VaR (95%)** | 單日風險值，統計極限損失 |

#### 5. **信號生成系統**
綜合技術面 + AI 預測生成 4 個等級信號：
- 🟢 **STRONG BUY**: 高信心買進
- 🔵 **BUY**: 適度買進  
- 🟠 **SELL**: 考慮減碼
- 🔴 **STRONG SELL**: 強勢賣出
- ⚪ **HOLD**: 觀望

## 📊 功能模塊詳解

### A. 數據獲取層
```python
fetch_stock_data(ticker, years)
```
**功能**:
- 自動識別台灣股票代碼 (.TW) 與美股代碼
- 下載 OHLCV (開高低收成交量) 歷史數據
- 自動計算 15+ 技術指標
- 緩存機制 (TTL=600秒) 降低 API 調用

**支援代碼格式**:
- 台灣: `2330`, `0050` (自動轉 .TW)
- 美股: `AAPL`, `TSLA`

### B. 技術指標計算

#### 布林通道
```
MA20 = 20 天移動平均
Upper = MA20 + 2 × 標準差
Lower = MA20 - 2 × 標準差
```

#### MACD
```
DIF = 12日EMA - 26日EMA
Signal = 9日EMA(DIF)
Histogram = DIF - Signal
```

#### RSI (相對強度指數)
```
RS = 平均上升幅度 / 平均下降幅度
RSI = 100 - (100 / (1 + RS))

判斷標準:
- RSI > 70: 超買區
- RSI < 30: 超賣區
- 30-70: 中立區
```

### C. 風險指標計算

#### 年化波動率
```
日報酬率標準差 × √252
(252 為交易天數)
```

#### 夏普比率
```
夏普比率 = (年平均報酬 - 無風險利率) / 年波動率
標準: 無風險利率 = 2%

解讀:
- >1.0: 優秀
- 0.5-1.0: 良好
- 0-0.5: 一般
- <0: 負值表現差於無風險資產
```

#### 最大回撤
```
尋找最高點後的最大跌幅
反映最壞情況下的損失幅度
```

### D. AI 預測模塊

使用 **Prophet** 時間序列模型:
- 自動分解趨勢、季節性、節假日效應
- 生成區間預測 (95% 置信度)
- 適合金融市場特性的參數設置

**預測步驟**:
1. 輸入歷史價格序列 (過去 3-5 年)
2. Prophet 自動擬合趨勢與季節性
3. 向前預測指定天數
4. 輸出點估計與置信區間

## 📱 使用指南

### 側邊欄配置

#### 基礎設定
- **Enter Tickers**: 輸入股票代碼 (逗號分隔)
  - 例: `2330,0050,2454` 或 `AAPL,TSLA`

- **Training Period**: 訓練數據年數 (1-10 年)
  - 推薦: 5 年 (平衡準確度與數據量)

- **Forecast Days**: 預測天數 (30-365)
  - 30 天: 短期交易
  - 90 天: 中期策略
  - 365 天: 長期配置

#### 高級設定
- **Confidence Level**: 置信度 (0.80-0.99)
  - 越高越保守
  - 推薦: 0.95

- **Min Volume Threshold**: 最小成交量
  - 過濾低流動性股票

- **Risk Tolerance**: 風險偏好
  - 影響信號生成的臨界值

### 主頁面解讀

#### 1. 股票概覽卡片
```
股票名稱 代碼
當前價格 ↑ 漲幅
```

#### 2. 信號區域
詳細的交易建議:
- 信號等級 (STRONG BUY / BUY / HOLD / SELL / STRONG SELL)
- 具體建議語句
- 信心度 (HIGH / MODERATE / NEUTRAL)

#### 3. 關鍵指標面板
5 個核心指標:
- **Technical Score**: 0-100 綜合評分
- **RSI (14)**: 0-100，判斷超買超賣
- **Volatility**: 年化波動率 (%)
- **Max Drawdown**: 歷史最大虧損 (%)
- **AI 預測**: 預期未來變動率 (%)

### 詳細分析標籤

#### 📈 技術面標籤
**雙層圖表**:
- **上層**: 價格 + 布林通道 + MA20 移動平均
- **下層**: 成交量柱狀圖

**信息提取**:
- 價格在布林通道位置 → 判斷超買超賣
- 成交量異常 → 確認趨勢
- MA20 方向 → 確認短期趨勢

#### 📊 動能分析標籤
**兩層子圖**:
- **上層**: MACD 柱狀圖 + DIF 線 + Signal 線
- **下層**: RSI 曲線 + 超買 (70) 超賣 (30) 線

**解讀技巧**:
```
MACD 柱狀圖:
- 紅色柱漸高 → 上升動能增強
- 綠色柱漸深 → 下降動能增強
- 柱大小縮小 → 動能減弱

RSI:
- 觸及 70 以上 → 可能見頂，考慮減碼
- 觸及 30 以下 → 可能見底，考慮買入
- 在 50 附近 → 市場處於平衡
```

#### 🔮 AI 預測標籤
**預測圖表**:
- 藍色線: 歷史價格
- 橙色虛線: AI 預測價格
- 灰色區域: 95% 置信區間

**預測統計**:
- **目標價**: 預期終點價格與漲跌幅
- **預測區間**: 樂觀案例 vs 悲觀案例
- **置信度**: 統計模型的信心水準

#### ⚠️ 風險報告標籤
**風險四大指標**:
1. **年化波動率**: 衡量價格變動劇烈程度
2. **夏普比率**: 判斷風險調整報酬是否優秀
3. **最大回撤**: 評估最壞情況損失
4. **VaR (95%)**: 單日損失極限

**風險評級**:
```
波動率:
- < 15%: 低波動 (藍籌股、ETF)
- 15-25%: 中等波動 (大型股)
- > 25%: 高波動 (小型股、新興)

夏普比率:
- > 1.0: 優秀的風險調整報酬 🟢
- 0.5-1.0: 良好
- 0-0.5: 一般
- < 0: 較差
```

### 組合對比

**適用於多股分析**:
- 表格展示各股關鍵指標
- 圖表比較技術評分
- 快速識別組合中的強弱股

## 🔧 進階設置

### 自訂技術評分權重

編輯 `calculate_technical_scores()` 函數:
```python
composite = (
    scores['trend'] * 0.35 +      # 趨勢權重
    scores['momentum'] * 0.30 +   # 動能權重
    scores['rsi'] * 0.20 +        # RSI 權重
    scores['volume'] * 0.15       # 成交量權重
)
```

### 調整信號臨界值

編輯 `get_signal_recommendation()` 函數:
```python
if composite_score >= 70 and ai_trend > 5:  # 調整這些數值
    # STRONG BUY...
```

### 修改預測模型參數

編輯 `forecast_stock_price()` 函數:
```python
m = Prophet(
    daily_seasonality=False,      # 是否包含日季節性
    yearly_seasonality=True,      # 年季節性
    weekly_seasonality=True,      # 週季節性
    interval_width=0.95           # 置信區間寬度
)
```

## ⚠️ 重要提醒

### 1. 數據延遲
- Yahoo Finance 數據通常延遲 15-20 分鐘
- 台灣股票數據可能不如 US 市場完整

### 2. 預測局限
- Prophet 基於歷史規律，無法預測黑天鵝事件
- 金融市場存在不可預測性
- 歷史表現不代表未來結果

### 3. 技術分析局限
- 技術指標滯後性
- 不同周期可能給出矛盾信號
- 應結合基本面分析

## 🐛 常見問題

### Q1: 「無法獲取數據」錯誤
**原因**: 
- 股票代碼拼寫錯誤
- 代碼不存在
- 網絡問題

**解決**:
```python
# 確保台灣股票代碼包含 .TW 字尾
# 或直接輸入 2330 (系統自動轉換)

# 測試連接:
import yfinance as yf
yf.download("2330.TW")
```

### Q2: 預測虛線不出現
**原因**: 數據點不足 (少於 20 筆)

**解決**: 增加 Training Period 至 3 年以上

### Q3: 信號與直觀判斷相反
**可能情況**:
- 技術面超買但 AI 看漲 (看波段頂部前反彈)
- 技術面超賣但 AI 看跌 (看底部確認前調整)

**處理**: 綜合多個信號，不盲目追隨單一指標
